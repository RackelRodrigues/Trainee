<h1 align="center">Trainee</h1>

## 📌 Sobre o projeto

O **Trainee** é uma plataforma web de candidaturas desenvolvida para conectar candidatos a oportunidades de trabalho, com foco inicial em estágios.

A aplicação simula um ambiente real de mercado, aplicando boas práticas de engenharia de software, organização de código, acessibilidade e automação de processos.

📖 Documentação completa:  
👉 [Acessar no Notion](https://hazel-drifter-7e9.notion.site/Trainee-33c635b2f92180d794c2f988306f7ceb)

---

## 🚀 Funcionalidades

- Criação e gerenciamento de perfis (candidato e empresa)
- Listagem de vagas disponíveis
- Candidatura a vagas
- Gerenciamento de candidaturas
- Interface responsiva
- Acessibilidade (a11y)

---

## 🧠 Tecnologias e Stack

### Front-end

- Next.js (App Router)
- React
- TypeScript
- React Hook Form
- Zod (validação)
- Sass (estilização)
- SEO básico

### Testes

- Jest + React Testing Library

### DevOps

- GitHub Actions (CI/CD)
- Vercel (deploy automático)

---

## 📁 Estrutura do projeto

```bash
src/
  app/            # rotas e páginas (App Router)
  components/     # componentes reutilizáveis
  hooks/          # hooks customizados
  services/       # integração com API
  utils/          # funções auxiliares
  styles/         # estilos globais
```

## ⚙️ Instalação e execução

```bash
git clone https://github.com/SEU_USUARIO/Trainee.git
cd trainee
npm install
npm run dev
```

Acesse:

```
http://localhost:3000
```
